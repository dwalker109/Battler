# Battler

Dan Walker, November 2016

# Install

Requires `composer`. Clone, then run `composer install`

# Usage

Make `./battler` executable:

    chmod +x ./battler
    
Run the simulation:

    ./battler
    
# Tests

Some simple tests are included - to run them:

    ./vendor/bin/phpunit

# Standards

Targets PSR2 via `php-cs-fixer`.
